Caisse Moroges - PWA

Pour l'installer sur iPad :
1. Mettre le dossier sur un hébergement web compatible HTTPS (GitHub Pages, Netlify...).
2. Ouvrir l'adresse dans Safari.
3. Bouton Partager > Ajouter à l'écran d'accueil.
4. L'application fonctionne ensuite en plein écran et hors connexion.

Google Drive n'est pas recommandé pour une PWA.
